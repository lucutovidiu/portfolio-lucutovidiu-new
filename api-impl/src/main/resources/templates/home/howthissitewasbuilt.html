How was this site built

This website is my digital canvas, where I showcase my present and upcoming applications, artfully designed for peak performance and contemporary tech relevance. Initially crafted with Next.js and React, it has recently undergone a transformative evolution, reborn with the prowess of Java Spring Boot and Thymeleaf.

The tech tapestry is rich and diverse: Thymeleaf, with select Javascript for rapid AJAX responses, weaves an engaging frontend. On the backend, the foundation is Java paired with Spring Boot 2.4.1, guided by Maven's adept dependency management. Security is fortified through Spring Security, offering multi-role access. Swagger enhances development, simplifying API documentation. Our data domain is underpinned by MongoDB, seamlessly entwined with Spring Boot via MongoRepository.

Yet, it's the innovative touches that truly set this site apart. It empowers me to zip my portfolios and download them as a coherent package, complete with project specifics stored in JSON files for effortless re-upload. All this, while automatically resizing, compressing, and optimizing images during portfolio uploads, ensuring peak website performance. And don't forget the dedicated email service, ready to facilitate user interaction on the contact page. It's a digital masterpiece in the making, blending the elegance of Thymeleaf and the potency of Spring Boot to redefine my online presence.
